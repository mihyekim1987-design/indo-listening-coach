# 🇮🇩 Belajar Bahasa Indonesia

AI 기반 인도네시아어 학습 앱 - Streamlit으로 제작

## ✨ 주요 기능

### 1. 🏠 메인 홈 화면
- 직관적인 카드 형태의 네비게이션
- 오디오/YouTube/텍스트 학습 간편 접근
- 최근 학습 통계 실시간 표시

### 2. 📊 CEFR 기반 난이도 선택
- A1 (입문): 기본 인사, 자기소개
- A2 (초급): 일상 대화, 기본 문법
- B1 (중급): 의견 표현, 복잡한 문법
- B2 (중상급): 추상적 주제, 고급 문법

### 3. 💪 컨디션별 문제 수 조절
- **A (여유)**: 10문항 - 충분한 시간으로 깊이 있는 학습
- **B (보통)**: 5문항 - 적당한 분량의 균형 잡힌 학습
- **C (힘듦)**: 3문항 - 핵심만 빠르게 학습

### 4. 📚 다양한 학습 콘텐츠
- **텍스트**: 위키문헌, VOA 뉴스 기사
- **유튜브**: VOA Indonesia 다큐멘터리, 뉴스 (자막 추출)
- **오디오**: Whisper ASR로 음성 인식 학습
- **직접 입력**: 원하는 URL 직접 입력 가능

### 5. 🎯 AI 퀴즈 생성 (Structured Outputs)
- GPT-4o가 콘텐츠 기반 퀴즈 자동 생성
- 카테고리별 분류 (어휘/문법/독해/청해/숫자/시간)
- **근거 원문 인용(evidence_quote)** 포함
- 상세 해설 및 보기별 설명 제공

### 6. 📄 임베드 미리보기
- 학습 전 원본 콘텐츠 임베드로 확인
- 텍스트 자동 추출
- 유튜브 영상 임베드 + 자막 추출

### 7. 🔄 틀린 문제 반복 학습 시스템
- 틀린 문제와 유사한 새로운 문제 자동 생성
- 정답할 때까지 반복 연습
- 진행률 실시간 추적
- 학습 결과 페이지 바로가기

### 8. 🗣️ TTS 섀도잉 연습 (edge-tts)
- **edge-tts 기반 자연스러운 인도네시아어 음성**
- 오답 근거 문장 자동 재생
- 속도 조절 가능 (느리게/보통/빠르게)
- 오디오 파일 캐싱으로 빠른 재생
- 한국어 fallback 제거 (인도네시아어 전용)

### 9. 🤖 AI 학습 코치
- 개인화된 학습 조언 생성
- 강점과 약점 분석
- 즉시 실행 가능한 액션 플랜
- 추천 학습 자료 제공

### 10. 📈 학습 결과 대시보드
- **취약 카테고리 분석** (클릭 가능한 카드)
- **카테고리별 학습 현황** (SRS 기반)
- 일별/주간 학습 통계
- 연속 학습일 추적
- JSON 파일 다운로드

### 11. 🔁 Spaced Repetition System (SRS)
- SM-2 알고리즘 기반 복습 스케줄링
- 카테고리별 마스터 진행률 추적
- 오늘 복습할 항목 자동 알림
- 정확도 기반 복습 간격 자동 조정

## 🚀 시작하기

### 1. 필요 조건
- Python 3.8 이상
- OpenAI API 키 (GPT-4o 접근 권한)

### 2. 설치

```bash
# 레포지토리 클론 또는 파일 다운로드
cd indonesian_app

# 가상환경 생성 (권장)
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 패키지 설치
pip install -r requirements.txt
```

### 3. API 키 설정

`.env.example` 파일을 `.env`로 복사하고 API 키를 입력하세요:

```bash
cp .env.example .env
```

`.env` 파일 편집:
```
OPENAI_API_KEY=sk-your-actual-api-key-here
HF_TOKEN=your-huggingface-token-here  # 선택사항
```

### 4. 오디오 파일 준비 (선택)

`indo-listening-coach` 폴더에 오디오 샘플 파일을 넣어주세요:
- `sample_A.wav`
- `sample_B.wav`

### 5. 실행

```bash
streamlit run app.py
```

브라우저가 자동으로 열리고 메인 홈 화면이 표시됩니다.

#### 첫 실행 시 안내
- **Whisper 모델 다운로드**: 첫 오디오 학습 시 자동 다운로드 (수 GB, 시간 소요)
- **TTS 캐시**: 섀도잉 연습 시 자동으로 `logs/tts_cache/` 폴더에 음성 파일 생성
- **학습 데이터**: 학습 기록은 `logs/` 폴더에 JSON 형태로 저장

- **Frontend**: Streamlit

- **TTS**: edge-tts (Microsoft Neural Voice)

- **유튜브**: youtube-transcript-api
- **스타일링**: Custom CSS
- **데이터**: JSON (로컬 저장)
- **학습 알고리즘**: SM-2 Spaced Repetition

## ⚠️ 주의사항

1. **API 키 보안**: `.env` 파일은 절대 GitHub에 업로드하지 마세요!
2. **Whisper 모델**: 처음 실행 시 자동 다운로드 (수 GB, 시간 소요)
3. **유튜브 자막**: 자막이 없는 영상은 텍스트 추출이 불가능합니다
4. **웹 스크래핑**: 일부 웹사이트는 텍스트 추출이 제한될 수 있습니다
5. **TTS 음성**: edge-tts는 인터넷 연결이 필요합니다 (첫 생성 시)
6. **브라우저 TTS**: Web Speech API fallback은 브라우저에 인도네시아어 음성이 설치되어야 합니다

## 💡 주요 기능 사용 가이드

### 📊 학습 결과 대시보드
1. 메인 홈에서 "📊 학습 결과" 카드 클릭
2. **대시보드**: 전체 학습 통계 및 취약 카테고리 분석
3. **반복 학습**: 틀린 문제를 모두 정복할 때까지 반복
4. **섀도잉**: TTS로 인도네시아어 발음 연습
5. **SRS 복습**: 오늘 복습할 항목 자동 추천

### 🗣️ 섀도잉 연습
1. 문제 풀이 후 오답 근거 문장이 자동으로 섀도잉 연습에 추가됩니다
2. 속도 선택 (느리게/보통/빠르게)
3. 재생 버튼 클릭하여 음성 듣기
4. 한국어 번역 확인
5. 따라 말하며 발음 연습

### 🔄 반복 학습 시스템
1. 문제 풀이 후 "🚀 틀린 문제 반복 학습 시작" 버튼 클릭
2. 틀린 문제를 하나씩 다시 풀기
3. 정답을 맞출 때까지 반복
4. "🔄 유사 문제" 버튼으로 추가 연습 가능
5. 모두 정복 후 학습 결과 대시보드로 이동

### 🎯 취약 카테고리 분석
1. 학습 결과 대시보드에서 "🎯 취약 카테고리 분석" 섹션 확인
2. 카테고리 카드 클릭하여 상세 정보 확인
   - 오답 근거 문장들
   - 추천 학습 활동
   - 카테고리 설명
3. 취약점을 파악하고 집중 학습
